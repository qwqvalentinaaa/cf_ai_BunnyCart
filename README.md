# AI Shopping Assistant

An AI-powered shopping assistant that helps users find, compare, and evaluate products based on their preferences. Built using Cloudflare Workers, Workflows, and a large language model (LLM), this application provides recommendations, top product rankings, and pros/cons summaries.

## Sample Questions

- "I want to buy a shirt under $1000."  
- "Highlight pros and cons of the products."

## Deployment

Try the AI Shopping Assistant online: [https://kittyshopper.vltnzhang.workers.dev/](https://kittyshopper.vltnzhang.workers.dev/)

## Local Setup

Clone the repo and run the following command.

```bash
npm install
npm start
```
Open browser and go to corresponding local host page.
